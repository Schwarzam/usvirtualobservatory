TAP service Build application:

Once all the required steps(System requirements,installation of softwares and updating properties file) are done. Go to the "code-distribution" directory where "Build.xml" is given. If you are familiar with ANT-build, edit this build.xml according to your requirements. 
Use proper commands to build the tapservice executable. (If you do not have Apache ANT installed use the package given in compiler distribution)

or use following commands.

cd  ..\code_distribution 
code-distribution> ..\apache-1.8..\bin\ant dist

it will create the dist folder which contains compiled ".war" file .
This "*.war" file is used to deploy on the web server.
