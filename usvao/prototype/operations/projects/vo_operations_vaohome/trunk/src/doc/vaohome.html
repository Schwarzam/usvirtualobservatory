USVAO Home Page and the Role of the VAO Monitor and Notification Systems

The USVAO Home Page at usvao.org displays the current status of each of the 4 primary Science Services. The status is represented by one of 3 possible icons. These icons are A) a green dot (service is up) B) a red X (service is down), and C) a yellow triangle (service has an active or pending notice). The local (HEASARC) system is responsible for creating these icons and it behaves in the following way:

1. A cronjob runs hourly that retrieves an xml page containing the latest status of every VAO service (not including Legacy services). This is done by querying the VAO Monitor with:

http://heasarc.gsfc.nasa.gov/vo/vaomonitor/vaodb.pl?format=xml

2. A cronjob runs hourly that retrieves an xml page containing any active or pending server notices entered by users.This is done by querying the notification system for all current or pending notices via:

http://heasarc.gsfc.nasa.gov/vo/notification/shownote.pl?format=rawcurrent.

The xml response is stored in a file in the /tmp.shared area for use by other applications. This tmp.shared area is a publicly writable area.

3. An update script runs hourly at the HEASARC that reads the xml files in (1) and (2) and generates soft links to the correct icons. Since this script is always reading the latest status and notices files, the status represented by the icons is reliable to within an hour.
