VAOMonitor Page

Overview

The Vaomonitor page contains the most recent status of every VAO service. The user should keep in mind that services are tested once per hour so the status is not a real time representation of the health of that service.

How the Page is Divided

The page is divided into 6 general sections. They are:

Section 1/2: Portal Services and Tools
These sections show the current status of the primary VAO services

Section 3: Non-VAO Failing Services. This section describes the status of services that are currently failing outside of the VAO. These tests are being handled by an underlying Nagios framework and the results are generated from the page:

http://heasarc.gsfc.nasa.gov/vo/monitor

This section only shows the services that are failing, not the ones passing.

Section 4:System These are system tools. E.g. The harvester tool and SIMBAD.

Section 5:Harvester
This is a pointer to a page that describes the last time data was harvested from the various VO data providers

Section 6: Selenium Tests
These are heavier tests that are being run 3 times per day (9:00a.m., 12:00p.m. and 3:00 p.m. EST). The results of these tests are not being stored in the database due to several bugs in the tests and in the setup configuration. However, they are relatively reliable and the results should be compared to the standard hourly test on a service. E.g. If the VO Inventory/Portal test is passing under the "Portal" category but the selenium test is not, then the service should be manually tested to confirm whether or not there is indeed an issue. More information on how to setup and run selenium tests for this system can be found on the Selenium Tests page.
