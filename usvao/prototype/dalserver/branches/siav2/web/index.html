DALServer Builtin Demonstration and Test Queries
DALServer Version 0.7 September 2014
Built upon the DALServer toolkit.

Builtin Test Queries

  • SCS Test Query (catalog cone search)

  • SIA Null/Test Query (image access)

  • SSA Null/Test Query (spectral access)

These demonstration and test queries are built into the generic framework and are available immediately after deployment of the DALServer web-app to verify correct functionality. Any actual data services may be configured once DALServer has been deployed.

SCS (Cone Search) Test Service

 (defaults to Messier catalog)
Search Region (Debug ):
RA (degrees): DEC (degrees):
SR (search radius degrees): VERB (table verbosity 1,2,3):
Available Tables: Messier catalog (builtin version)
Output Data Formats: HTML Text CSV VOTable FITS

SIA Test Query (Legacy SIAV1)

 (Null/Test Query)
Search Region (Debug ):
POS ("ra,dec" in degrees): SIZE (search box size in degrees):
FORMAT (all,fits,graphic): VERB (table verbosity 0,1,2,3):
Image Data Collections: Null/Echo Test
Output Query Response Formats: HTML Text CSV VOTable FITS

SSA Test Query

 (Null/Test Query)
Search Region (Debug ):

SPATRES (min spatial resolution): TIMERES (min time resolution):
REDSHIFT (redshift range): VARAMPL (variability range):
TARGETNAME: TARGETCLASS:
