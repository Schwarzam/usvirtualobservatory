SLAP Parameters
VERSION[STRING]SLAP protocol version
REQUEST[STRING]Operation to be performed
WAVELENGTH[FLOAT, ORDERED, RANGELIST]Range of allowable line wavelengths (meters)
CHEMICAL_ELEMENT[STRING, RANGELIST]Desired chemical elements
INITIAL_LEVEL_ENERGY[FLOAT, ORDERED, RANGELIST]Energy range for initial level of transition (Joules)
FINAL_LEVEL_ENERGY[FLOAT, ORDERED, RANGELIST]Energy range for final level of transition (Joules)
TEMPERATURE[FLOAT]Temperature of object (Kelvin)
EINSTEIN_A[FLOAT, ORDERED, RANGELIST]Range of allowable transition probabilities (s^-1)
PROCESS_TYPE[STRING]Physical process type
PROCESS_NAME[STRING]Physical process which generated this line
FORMAT[STRING]Desired output data format
Maxrec[INTEGER]Maximum number of output records
Compress[BOOLEAN]Allow dataset compression
RunID[STRING]Runtime job ID string
