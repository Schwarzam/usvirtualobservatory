SIAP Parameters
VERSION[STRING]SIAP protocol version
REQUEST[STRING]Operation to be performed
POS[FLOAT, RANGELIST]Central coordinates of search region
SIZE[FLOAT]Size (diameter) of the search region
FORMAT[STRING, RANGELIST]Allowable output data formats
INTERSECT[STRING]Specifies how image footprint may overlap ROI
NAXIS[INTEGER, RANGELIST]Size of the output image in pixels
CFRAME[STRING]Spatial reference frame of output image
EQUINOX[FLOAT]Equinox of spatial coordinate frame
CRPIX[FLOAT, RANGELIST]Pixel coordinates of reference point
CRVAL[FLOAT, RANGELIST]World coordinates of reference point
CDELT[FLOAT, RANGELIST]Output image scale in degrees per pixel
ROTANG[FLOAT]Rotation of output image relative to CFRAME
PROJ[STRING]Celestial projection of the output image
Maxrec[INTEGER]Maximum number of output records
Compress[BOOLEAN]Allow dataset compression
RunID[STRING]Runtime job ID string
