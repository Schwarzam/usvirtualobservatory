Table Reformat and Validation Service

 Apply name resolution/coordinate transformation.

     

Upload your table file here, and this service will attempt to reformat and validate it to ensure compliance with specific formatting rules, and identify potential errors. If the validation is successful, download the reformatted table as an ASCII or VOtable file. You may then upload it to the desired IPAC or VO service.

If you check the name resolution/coordinate transformation box, the service will try to resolve source names using NED and SIMBAD, and to identify coordinate columns.

For more information, see: Using Tables With IPAC Services
