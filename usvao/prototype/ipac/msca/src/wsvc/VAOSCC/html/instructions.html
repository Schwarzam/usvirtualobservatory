VAO Cross-Comparison


positionally cross-correlates user tables with spatially-indexed archive catalogs. Both matches and unmatched input are returned.

Basic workflow:

  • Upload tables into the list below, then
  • Select one User Table and one On-line
    Catalog from the lists,
  • Set the maximim matching distance, and
  • Click Compare for processing.

Results appear on the right, with links to downloadable data files.

Previous runs can be recalled by selecting from the processing History.

For more information, consult the User Guide. For best results, use a newer version of the Firefox Web browser.

For other VAO services and information on the VAO itself, see the VAO Home Page.
