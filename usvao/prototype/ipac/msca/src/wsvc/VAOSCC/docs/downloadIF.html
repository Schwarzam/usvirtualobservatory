HTTP Interface: Data Retrieval (Table Download)

This service is used to retrieve result tables. The base URL for the retrieval service is:

	   http://vao-web.ipac.caltech.edu/cgi-bin/VAOSCC/nph-vaosccDownload

and there are two parameters:

  • file.   The desired result file. See the return from the cross-comparison service for a list of available files.
  • format.   The desired return format, which can be VOTable, IPAC/ASCII, FITS, CSV, TAB or BAR. The last three are comma-, tab-, and vertical-bar-delimited ASCII.

The return is the data itself as an HTTP message (with header including proper MIME type).
