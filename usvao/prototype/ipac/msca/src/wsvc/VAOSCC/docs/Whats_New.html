VAO Cross-Comparison Tool: What's New?

Overview

This service performs fast positional cross-matches between an input table of sources and common astronomical source catalogs. The example used here is to cross-compare a table of sources extracted from the WISE preliminary data release catalog with the SDSS DR9 database, in an effort to identify new brown/dwarf candidates. The service does not return cross-identifications, but rather cross-identification candidates. That is, it returns a table of all SDSS DR9 sources within a user-specified radius of each WISE source. See the User Guide for details.

Whats New

15 Apr 2013 This release includes one new catalog: SDSS DR9 (1233343212 records). This is a superset of and replaces SDSS DR7.

30 Nov 2012 This release includes new catalogs, pointers to catalog documentation, and beta release support for SAMP message passing. A minor bug was fixed which prevented some data downloaded through the VAO Data Discovery tool to be ingested properly.

In addition to the catalogs in the first release (IRAS, SDSS DR7, 2MASS, USNO B1, and WISE) we have added DENISD3, UCAC, Tycho2 and PPMXL. Work is continuing to expand this list to at least all large catalogs.

The Cross-Comparison service is not a source for any of the catalogs it has indexed but users still need access to catalog documentation in order to interpret their results. Rather that replicate this documentation, we provide links to primary or other on-line catalog descriptions.

The Simple Application Messaging Protocol (SAMP) is used to allow client tools and server applications through AJAX interfaces to send messages to each other. These messages most often point to data. For the Cross-Comparison service this is used in two places. First, another service (such as Topcat or the VAO Data Discovery Tool) can broadcast a message pointing to a table which the Cross-Commparison service then retrieves and presents in the list of uploaded files. The user can then compare this data to any of the indexed catalogs. Second, any of the results tables (matches, unmatched sources) can be broadcast by the Cross-Comparison service and picked up by other tools like Aladin or Topcat.
