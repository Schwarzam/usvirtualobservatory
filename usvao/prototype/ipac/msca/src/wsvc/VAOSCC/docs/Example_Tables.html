Example Input Data Files

Here is a set of example input files for this tool. For more information on formats,
see the Data Formats documentation.


IPAC_ASCII_one_header.tbl Simple column-aligned ASCII with a header. 10 sources
csv_table_example.tbl Same data, comma-separated text. 10 sources
tab_delimited_table_example.tbl Same data, tab-delimited. 10 sources
list_of_obj_names.tbl A mixed set of objects identified by name and cordinate strings. 10 sources
Abell.tbl The Abell galaxy cluster catalog. 2712 sources
AAVSO.xml The AAVSO Variable Star Index
(as downloaded from VizieR in VOTable format).		 203354 sources

You can download any of these files and send them to the Cross-Comparison Tool.
