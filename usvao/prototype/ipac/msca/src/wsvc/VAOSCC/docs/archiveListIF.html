HTTP Interface: Listing Available Catalogs

There is a very simple interface to aid in getting a list of archived catalogs. The base URL for the listing service is:

	   http://vao-web.ipac.caltech.edu/cgi-bin/VAOSCC/nph-vaosccArchiveList

The return structure from the listing service is simple XML (currently no schema):

      <results>

	 <catalog>
	    <name> IRAS_PSC </name>
	    <nrec> 245889 </nrec>
	    <ncol> 81 </ncol>
	    <size> 145858755 </size>

	 </catalog>

	 <catalog>
	    <name> SDSS_DR7 </name>
	    <nrec> 453846233 </nrec>
	    <ncol> 95 </ncol>

	    <size> 558684717759 </size>
	 </catalog>

	 <catalog>
	    <name> TWOMASS_PSC </name>
	    <nrec> 470992970 </nrec>

	    <ncol> 127 </ncol>
	    <size> 594864126361 </size>
	 </catalog>

	 <catalog>
	    <name> USNO_B1 </name>

	    <nrec> 1045175762 </nrec>
	    <ncol> 56 </ncol>
	    <size> 538265519689 </size>
	 </catalog>

      </results>
