readdata(search_params)
Creates a new readdata object for displaying a VOTABLE.
Parameters:
search_params
{String} Parameters to be passed to the votable object. One of search_params.form, search_params.stream or search_params.url must be specified. The search_params.output parameter is required.
search_params.form Optional
{String} Name of the form used for specifying the VOTABLE.
search_params.searchparam Optional
Name of input element containing the VOTABLE. Required if search_params.form is specified.
search_params.url Optional
{String} URL pointing to a VOTABLE.
search_params.stream Optional
An input stream over which will be read a VOTABLE.
search_params.filename Optional
{String} Optional name of the VOTABLE file, used only for display purposes (displayed in parentheses after the title).
search_params.output
{Object} DOM element where the VO table will be displayed.
search_params.titletext Optional
{String} Text displayed for the title of the votable.
search_params.queryTitle Optional
{String} Text displayed (in parentheses) for the browser window title.
search_params.xsltdir
{String} Directory on the server to look for XSLT files.
search_params.postInput Optional
Content for a http post request for loading the VOTABLE.
search_params.updateCallback Optional
{function} Function to call whenever the VOView display is updated.

columnFormat(column, format)
Add additional formatting to a column. The function can be called when formatting the column, which can be used for formatting other parts of the road as well, e.g. an empty column at the beginning of the table if it exists.
Parameters:
column
{String|integer|Regex} Column to be formatted. This can be specified either as: 1) an integer (the column number in the original order); 2) a string matching substring like the column name; 3) a regular expression matching the column name.
format
{String|function} The formatting information for the column. This can be either: 1) as string, in which case it will replace the current value of the column. If the string contains "@@", it will be replaced by the current column value; 2) a function, in which case the function will be called with the DOM element of the table cell as its only argument.
displayPrefixColumn(display)
Display an empty column at the beginning of the table. This column can be filled in by the application user.
Parameters:
display

displaySelectColumn(display)
Display a column at the beginning of the table with checkboxes for selecting rows. This also activates the other aspects of VOView for tracking and manipulating row selection. This also implies that displayPrefixColumn is set to true.
Parameters:
display
{boolean} If true, display the column.
