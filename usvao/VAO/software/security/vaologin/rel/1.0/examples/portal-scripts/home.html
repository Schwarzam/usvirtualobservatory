Test activities

Testing Portal Functions

These are activities that are typical for a user interacting with a portal that supports logins.

  • Create a VAO login from simple portal:
    • Click on the "Register" link in the side panel.
    • Create an account:
      • after filling in the registration page, you should eventually get an email with a confirmation code; follow it to complete the reservation
      • after confirming your account, you should get another email confirming the completion of registration.
    • You should next see a login page; log in.
    • You should then be returned to this page and see that you are logged in.

  • Test the case where the portal wants to register users, too. Click on one the links below to register locally:

    We support VAO Logins.

    If you do not have a VAO account, Get one now.
    If you already have one, register now to use this site.

Testing the VAO Identity Portal

Visit the VAO Logins web site and click on "Profile and Preferences". Test the various services provided there, including:

  • requesting a password reminder
  • change your user profile (your fname, email, institution, etc.)
  • change you privacy settings.
