Welcome to the VAO Login Services Portal

This is the site where you can create VAO Identity--a login account that can be used any VAO-compatible portal.

If you've used the Virtual Observatory, then you know that much of the data and capabilities can be accessed anonymously through public services and web sites with out any special logins; however, some features and services require more security. This portal allows you to create and manage your own VAO Logon--a single unique identity which you can use to log onto any VAO-compliant portal. Get one here!

Two-minute FAQ:

What is a VAO Logon
Also called your VAO Identity, it is a user name and password that you can use to log into web portals in the VO.
Why create a VAO Logon?

Because your VAO logon is supported at a number of VO portals, you will now have fewer usernames and passwords to keep track of.

Your VAO login name gives you added power while working in the VO through its single sign-on capabilities--once you're logged into the VAO, you can move freely between portals without retyping you password. More importantly, portals and services can pass private data back and forth securely with each other on your behalf.

How do I get a VAO Logon?
You can create your logon by filling out the VAO registration form. We'll then send you a quick email confirmation which you must respond to so that we know you are contactable and not a robot. Once that is done, you'll set your username and password, and you're done.
