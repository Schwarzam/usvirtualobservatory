You can create and download a USVAO credential (X.509 certificate and private key) that can be loaded directly into applications that run directly on your machine—such as a web browser. This will be a so-called "end-entity" certificate (i.e. not a proxy) and an associated private key that will together be packaged in either the PKCS12 or PEM format.


Lifetime:    The time before the certificate expires
Package Key:       A password for sealing the credential file that will be downloaded. You will need this when you load the certificate into your application.
Format:    Preferred by browsers and Java.
   Commonly used by grid computing tools.
Account Password:       The password to your account
