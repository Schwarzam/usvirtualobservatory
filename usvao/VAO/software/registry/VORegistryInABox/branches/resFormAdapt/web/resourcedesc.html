To help you choose which resource you want to add to the NVO Registry, here are descriptions of the resources recognized by this entry form.

Organisation A named group of one or more persons brought together to pursue participation in VO applications.
Data Collection A logical grouping of data which, in general, is composed of one or more accessible datasets.
Project An organisation with a focused set of data and services that it maintains.
Authority A naming authority; an assertion of control over a namespace represented by an authority identifier.
Browser-Based Service A (form-based) interface intended to be accessed interactively by a user via a web browser.
CGI-Get
Service 		This service type is similar to a Cone Search or SIA in that parameters are provided as keyword-value pairs appended to a base URL.

A CGI-Get Service may or may not pertain to the sky; if it does, it is known as a Sky Service. If the output of a Sky Service is tabular in form, it is known as a Tabular Sky Service.
GLU Service A web-based service that is described in a GLU registry.
Conesearch A VO standard service intended for searching catalogs by sky position.

The standard interface to access a Conesearch, encoded in the URL, contains a right acsension, a declination, and a search radius--all in degrees (J2000).
SIA Service A VO standard service intended for search for and retrieve images.

SIA uses HTTP GET queries to encode query arguments in the request URL. It specifies a number of optional parameters that can control what images are returned, how cut-outs are to be made, and what formats to provide. It also allows the data provider to support non-standard arguments.
Web Service A SOAP-Based service using HTTP POST that is describable by a WSDL document.
Generic Service A resource that can be invoked by a client to perform some action on its behalf.

This resource type should only be used if none of the other services listed apply.
Generic Resource Any entity that is describable and identifiable by a IVOA Identifier.

This resource type should only be used if none of the other resources listed apply.
