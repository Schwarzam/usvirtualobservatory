Simple Query lets you search catalogs and data collections for sources or observations within a given radius of a sky position. Small (few thousand row) tables can be retrieved in their entirety with the All-sky option. !$IVOID You will need to enter an 'IVO identifier' for the catalog or collection you are querying. You can find these in the Directory.

To view an existing VOTable use our Table Viewer.
