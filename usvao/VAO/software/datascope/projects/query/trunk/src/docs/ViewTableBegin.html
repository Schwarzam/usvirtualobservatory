The NVO Table Viewer lets you display the columns and rows of a VOTable. You can sort and filter the table interactively and send the filtered results to other data discovery tools. The Table Viewer's capability for interactively viewing VOTables is used by the registry, Simple Query and the Inventory tools.

To query VO-enabled catalogs and data resources use our Simple Query tool.
