Search for Bautz-Morgan type II Abell clusters near the South Pole.

  1. In the registry query ask for 'Abell'
  2. The first entry is the Abell cluster of galaxies.
  3. Click on Search Me
  4. Enter 0.,-90 in the center and 20 in the search radius.
  5. Enter II in the Bautz-Morgan type and filter.
  6. There are 42 results.
Alternatively we could done an all-sky search and then filtered for dec<-70.
